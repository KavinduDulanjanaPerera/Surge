Answers to Questions and Screenshots are avalable in the "Surge Global Answers & Read Me File" document.

This is a simple user signup application where the user enters the details and their data will be
stored in a database. This app is connected to the Mongodb database by storing the details in a .env file.
The Passwords that the user submits through this form will be salted and hashed to minimize vulnerabilities.
Authentication using JWT web tokens was implemented but has been removed due to an error.
